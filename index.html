Briggs' HTML Cheatsheet

This Cheatsheet is designed to help remind the user of the basic boilerplate code for HTML and CSS

HTML

The first few lines

Tag Name Description
<!DOCTYPE html> Document Type Written on line 1, this tag informs the browser that the file is HTML.
<html> Code to be parsed Everything inbetween the html tags will be interprested by the browser
<head> Document metadata This is where the pages metadata is stored.
e.g. <title> will display the text between tags in the browser tab.
<link rel="stylesheet" href="URL" type="text/css"> will apply the CSS format of the linked file.

The main event

Tag Name Description
<header> Document header Identifies the code between tags as the document header
<body> Document body Identifies the code between tags as the body of the document
<footer> Document footer Identifies the code between the tags as the document footer

CSS

Targeting

Specificity Target Code
Highest - 1 ID Selector #example
2 Class selector .example
Lowest - 3 Type selector h1

Styling

Below is a few examples that are included in the CSS for this page

Target Code Description
html {
background-color: snow;
font-family: Helvetica, Arial, sans-serif;
} 		Sets the background colour for the entire page
Sets the font *multiple fonts are included to increase capatability*.
h1 {
background-image: url(./file in root);
color: powderblue;
font-size: 60px;
text-align: center;
text-shadow: -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;
} 		Sets the background image to a file in the .root directory.
Sets the colour of the font.
Sets the size of the font.
Aligns the font on the page.
Adds a border to the text.
h2 {
background-color: powderblue
} 		Sets the background colour.
table {
border-collapse: collapse;
} 		Collapses the gap between the <thead> and <tbody> sections.
thead {
background-color: skyblue;
text-align: center;
} 		Sets the background colour.
Aligns the text
tr tr:hover {
background-color: powderblue;
} 		Sets the background colour to change when the mouse hovers over the row.
td {
border: 2px solid grey;
padding: 4px;
} 		Applies a border to every cell.
Pads the top and bottom of the text in every cell.
.class {
background-color: dimgray;
color: white;
font-family: Georgia, 'Times New Roman', Times, serif;
} 		Styles the selected class in a particular way.
#id {
font-size: 40px;
font-family: cursive;
font-style: italic;
color: antiquewhite;
text-shadow: -1px 0 black;
} 		Styles the selected id in a particular way.
